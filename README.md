# ReactIQ ⚛

A React JS quiz app with real-time leaderboard.

## 🔗 Live Demo
https://react-quiz-steel.vercel.app/

## ✨ Features
- 10 React JS questions
- 15 second timer per question
- Real-time leaderboard via Firebase Firestore
- Dark mode toggle
- Responsive design

## 🛠 Tech Stack
- React JS
- Tailwind CSS
- Shadcn UI
- Firebase Firestore
- Vite
- Vercel
